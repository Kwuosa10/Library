#Library App

This project is a simple web application that lets users store and manage books in a personal library. Users can add new books remove books and change the read status of any book.

The goal of the project is to practice JavaScript constructors page rendering and basic interaction with forms.

#Features

Add a new book with a title author pages and read status
View all books in a clean display
Remove a book from the library
Toggle the read status of a book
Each book receives a unique id for easy identification

#Technologies Used

HTML
CSS
JavaScript

#How to Use

Open the project in your browser
Click the New Book button to show the form
Fill in the book details and submit
The book will appear on the page
Use the buttons on each book to remove it or change its read status
